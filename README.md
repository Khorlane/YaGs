# YaGs
Yet Another Game Server

The intent of YaGs is to give those that are interested in creating a MUD from scratch a chance to have that experience with some guidance. Once you have followed along with the development of YaGs, you can then decide if you want to take on the development of a more full featured complex MUD. It is doubtful that YaGs is a good foundation for your MUD, but there might some aspects of the design and code that you find useful.

The releases represent milestones as YaGs progresses from start to finish. The ability to compare releases, enables those who may be interested, to follow along as major blocks of functionality are added to the codebase. Comparing one commit with another might not be as instructive due to the path from one release to the next might not being very straight :)

Check out http://yags.wikidot.com/ for more information.
